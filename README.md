# Note App (Clean Architecture)
## This project was developed with the Kotlin programming language and the following technologies were used:<br/>
- Navigation Component <br/>
- Room database <br/>
- Coroutine <br/>
- Flow <br/>
- ViewBinding <br/>
- DataBinding <br/>
- Dagger (Hilt) <br/>
